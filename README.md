# 📝 Notix

A fast, beautiful, and minimal note-taking extension built directly into the Firefox Sidebar.

![alt text](https://github.com/Arvanta/Notix/blob/main/notix_ScreenShot.png?raw=true)

## 📦 Installation

* **Firefox:** Install the extension from the official Mozilla site: [Get Notix for Firefox](https://addons.mozilla.org/en-US/firefox/addon/notix/)
* **Chrome:** The version for Chrome and Chromium-based browsers will be released in the future.

## ✨ Overview

Notix offers a distraction-free environment to capture your ideas without leaving your current webpage. With a focus on UI/UX, privacy, and smooth interactions, it is the perfect lightweight alternative to heavy note-taking apps.

## 🚀 Features

* **Rich Text Formatting**: Bold, Italic, Underline, Lists, Alignments, and Links.
* **Smart Color Application**: Native color pickers that only affect selected text, preventing unwanted color inheritance.
* **Dynamic Theming**: Flawless Light/Dark mode support, including dynamic toolbar and sidebar icons.
* **Note Pinning**: Keep important notes at the top of your list.
* **Context Menu Integration**: Capture text from any webpage with a simple right-click.
* **Fully Local & Secure**: Zero cloud dependencies. All data is stored securely in Firefox local storage with XSS protection.
* **Import/Export**: Backup and restore your notes effortlessly via JSON.
* **Multilingual Friendly**: Built-in RTL/LTR text direction controls.
* **Keyboard Shortcuts**: Fully customizable and intuitive shortcuts for power users.
